Transfer Learning-Based Classification of Poultry Diseases

This project aims to classify poultry diseases into Salmonella, New Castle Disease, Coccidiosis, and Healthy using transfer learning. Farmers use a mobile app to input data (symptoms, conditions, images) and get an instant diagnosis with treatment suggestions. The goal: reduce disease impact, improve productivity, and empower rural communities and commercial farms.
